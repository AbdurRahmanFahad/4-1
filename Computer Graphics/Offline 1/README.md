
# Assignment on OpenGL

## Fully Controllable Camera (Problem 1.exe)

`Up arrow` - move forward
`Down arrow` - move backward 
`Right arrow` - move right
`Left arrow` - move left
`PgUp` - move up
`PgDn` - move down
`1` - rotate/look left
`2` - rotate/look right
`3` - look up
`4` - look down
`5` - tilt clockwise
`6` - tilt counterclockwise
